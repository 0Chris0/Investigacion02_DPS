## Requisitos Previos
* **Node.js**: Versión estable recomendada (LTS).
* **Expo Go**: Instalado en tu dispositivo móvil o configurado en un emulador.

Instalación y Ejecución

### 1. Instalar las dependencias 

1. **Instalar:** `npm install` Dependencias Base
2. **Instalar:** `npx expo install @viro-community/react-viro @reactvision/react-viro ` Dependencias de Viro y AR 
3. **Instalar:** `npx expo install react-native-safe-area-context react-native-screens ` Dependencias de navegacion y graficos
4. **Ejecutar:** `npx expo start -c`

*Si el teléfono no conecta bien, intenta con:* `npx expo start -c --lan` (Esto funciona como una red local para conectar mejor el celular).
